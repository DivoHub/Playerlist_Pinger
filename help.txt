Commands
-----------------------------

start
    -start checker

stop
    -stop checker

exit
    -exit application

addplayer
    -append additional players to checking list

delplayer
    -delete specified player from checking list

onlinenow
    -print current list of players online

changeint
    -change interval (seconds) between each online check

changeserver
    -change server IP to be checked

checkconfig
    -displays current config (players list, server ip, interval)

fresh
    -reinitialize, and create new config file

help
    -displays command list
